Qu'est-ce qu'un pipe ?

Un pipe en Angular permet de transformer les données avant de les afficher dans la vue. Ils facilitent le formatage des données sans modifier le modèle d'origine.

Utilisations

Utilisation sans paramètre

Pour utiliser un pipe, il suffit de le placer dans une expression entre doubles accolades {{ }} suivis de | dans le HTML. Par exemple, pour afficher une date formatée, on peut utiliser le pipe date :
{{ maVariableDate | date }} => {{ maVariableDate | date }}
Sans pipe: {{ maVariableDate }}

Utilisation avec paramètre

Certains pipes acceptent des paramètres pour personnaliser leur comportement. Pour cela, on ajoute un : suivi du paramètre souhaité. Par exemple, pour afficher une date en français, on peut utiliser le pipe date avec le paramètre 'shortDate' :
{{ maVariableDate | date:'shortDate' }} => {{ maVariableDate | date: "shortDate" }}
Sans pipe: {{ maVariableDate }}

Pipes personnalisés

Il est possible de créer ses propres pipes pour répondre à des besoins spécifiques. Pour cela, il suffit de créer une classe TypeScript avec le décorateur @Pipe et d'implémenter l'interface PipeTransform. Ensuite, il faut définir la méthode transform qui prend en paramètre la valeur à transformer et les paramètres optionnels. Enfin, il faut l'ajouter dans le module de l'application pour pouvoir l'utiliser.

  1. Générer un pipe : Utilisez Angular CLI pour générer un pipe.
    ng generate pipe customPipeName
  2. Analyse
    • name : Nom du pipe utilisé dans le template.
    • transform : Méthode de transformation.
    • value : Valeur à transformer.
    • args : Paramètres optionnels.
  3. Implémentation : Écrivez la logique de transformation dans le pipe.
