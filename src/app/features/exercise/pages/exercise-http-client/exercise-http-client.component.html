Avant de commencer cet exercice

Lance l'API json-server qui se trouve dans ce projet avec
npm run fakeApi

Bibliothèque

Fichier de travail :
src/app/features/exercise/pages/exercise-http-client/components/book

Crée une mini gestion de livre dans le composant spécifié ci-dessus en utilisant les concepts vues dans ce chapitre et les chapitres précédents et en utilisant json-server comme API fictive.

⚠️ La collection books existe déjà dans le json-server, regarde le fichier avant de commencer!

Un livre devra possèder les informations suivantes:

  • un id (id auto-générer et incrémenter)
  • un isbn (isbn)
  • un titre (title)
  • un auteur (author)
  • une description (description)
  • une date de publication (releaseDate)

Fonctionnalités:

  • Afficher la liste des livres
  • Ajouter un livre
  • Supprimer un livre
  • Modifier un livre

Contraintes:

  • Travailles avec ReactiveForms pour les formulaires
  • Travailles avec plusieurs composants (composant de listing, composant d'ajout...)

Bonus:

  • Gérer la pagination du listing
  • Utiliser une modale pour l'ajout et la modification d'élément
