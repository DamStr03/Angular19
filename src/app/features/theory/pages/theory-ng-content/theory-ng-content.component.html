Introduction :

NgContent est une directive Angular qui permet d'insérer du contenu HTML dans un composant. Elle est utilisée pour créer des composants réutilisables et flexibles en permettant aux développeurs de définir des emplacements dans le modèle du composant où le contenu peut être inséré.
