Voici un exemple de code qui se trouverait dans un fichier de constants api.

Le fait de regroupé les constantes dans un fichier permet de les centraliser et de les réutiliser plus facilement.

De plus, si ces mêmes valeurs sont utilées à plusieurs endroits, il sera plus facile de modifié de uniquement modifié ce fichier plutôt que tout les endroits où est utilisé cette valeur.
