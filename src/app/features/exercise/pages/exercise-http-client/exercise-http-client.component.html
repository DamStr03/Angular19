Avant de commencer cet exercice

Lance l'API json-server qui se trouve dans ce projet avec
npm run fakeApi

Bibliothèque

Fichier de travail :
src/app/features/exercise/pages/exercise-http-client/components/book

Crée une mini gestion de livre dans le composant spécifié ci-dessus.

Un livre devra possèder les informations suivantes:

  • un id (auto-générer)
  • un nom
  • une description
  • un auteur
  • une date de publication

Fonctionnalités:

  • Afficher la liste des livres
  • Ajouter un livre
  • Supprimer un livre
  • Modifier un livre

Contraintes:

  • Travaillé avec ReactiveForms pour les formulaires
  • Travaillé avec plusieurs composants (composant de listing, composant d'ajout...)

Bonus:

  • Gérer la pagination du listing
  • Utiliser une modale pour l'ajout et la modification d'élément
