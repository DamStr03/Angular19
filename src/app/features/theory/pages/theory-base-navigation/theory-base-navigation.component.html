Précédemment, nous avons vu comment préparer le system de navigation de notre application mais nous n'avons pas encore utiliser les routes pour naviguer entre les différentes pages de notre application en SPA (Single Page Application).
Nous devions passer par la barre d'adresse pour naviguer entre les pages de notre application.

RouterLink

En HTML, pour naviguer entre les différentes pages d'un site web, nous utilisons les balises <a href="">.
Sauf que l'utilisation de href recharge la page à chaque fois que l'on clique sur un lien.
Exemple: Chapitre 2 via href

Pour naviguer entre les différentes pages de notre application sans recharger l'application, nous allons remplacer href par routerLink.
Exemple: Chapitre 2 via routerLink

routerLink est une directive qui permet de naviguer entre les différentes pages de notre application sans recharger l'application.
⏱️les directives seront abordées plus tard

Par exemple pour se rendre sur la page de théorie actuelle, nous avons le code suivant: Chapitre 2 via routerLink

⚠️ pour utiliser la directive routerLink, il ne faut pas oublié de l'importer dans le composant.
