Utilise tout ce que tu as appris dans le chapitre des pipes!

Objectif

Crée plusieurs pipes pour transformer des données.

Exercie 1 - UnE LeTtRe SuR 2.

Créer une pipe pour transformer un texte en alternant une lettre sur deux en majuscule.

Exercie 2 - Affichage de tableau

Créer une pipe qui prendra un paramètre pour afficher un tableau de données.
Par exemple:

  • Par défaut: ['a', 'b', 'c'] deviendra a, b, c
  • Avec un paramètre qui serait renseigner:
    • (-) : ['a', 'b', 'c'] deviendra a - b - c
    • (+-) : ['a', 'b', 'c'] deviendra a +- b +- c
    • (...) : ['a', 'b', 'c'] deviendra a ... b ... c
    • ... Bref, l'utilisateur choisit.
