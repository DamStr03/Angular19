Démo - Composant

Animalerie

Le but de cet exercice est de créer une animalerie.

  • Dans le composant animalerie créer un dossier components
  • Dans le dossier components créer:
    • un composant chat et customisé son contenu
    • un composant chien et customisé son contenu
  • Afficher les composants chat et chien dans le composant animalerie
