Contexte:

Nous n'avons, pour le moment, utilisé que la partie HTML et CSS de nos composants Angular. Voyons maintenant, à travers différents moyens, comment utiliser la partie TypeScript pour nous aider dans le développement de notre application !

Objectif:

Utiliser le binding pour intégrer du TypeScript dans la partie HTML de nos composants et automatiser certaines tâches tout en optimisant la gestion de l'interface utilisateur.

Le binding:

Le binding (liaison) en Angular fait référence à la synchronisation des données entre le modèle (TypeScript) et la vue (HTML). Il existe différents types de binding :

  • Interpolation: Elle permet d'insérer des expressions TypeScript directement dans le HTML en utilisant les doubles accolades {{ et }}.
  • Property Binding: Il lie les propriétés du DOM aux propriétés des composants TypeScript à l'aide de crochets [].
  • Event Binding: Il permet de lier des événements utilisateur (comme un clic) à des méthodes TypeScript en utilisant des parenthèses ().
  • Two-way Binding: Il combine les deux et permet une synchronisation bidirectionnelle entre le modèle et la vue via [(ngModel)].

Rappel - le DOM

Le DOM (Document Object Model) est une représentation en mémoire de la structure d'une page web. Il permet à JavaScript (ou TypeScript) d'interagir avec les éléments HTML et de les manipuler dynamiquement.

En Angular, le DOM est souvent modifié à travers le binding, où les changements de données dans les composants peuvent automatiquement se refléter dans l'interface utilisateur sans qu'il soit nécessaire de recharger la page.

Exemple - Interpolation (one-way binding)

Afficher une variable dans l'HTML

Dans le fichier TypeScript:
title = "Bienvenue sur Angular";

Dans le fichier HTML:
{{ title }}

Résultat : {{ title }}

Exemple - Property Binding

Lier un attribut HTML au TS

Utilisation du property binding pour lier une propriété HTML à une variable TypeScript :
estDesactive = true;

Dans le fichier HTML:
<button [disabled]="estDesactive">je suis désactivé</button>

Exemple - Event Binding

Lier un évenement à une méthode TS

Dans le fichier TypeScript :
handleClick() { alert("Button clicked!"); }

Utilisation de l'event binding pour réagir aux événements utilisateur :
<button (click)="handleClick()">Clique-moi !</button>

Exemple - Two-way Data Binding

Récupérer & Mettre à jour une valeur TS depuis l'HTML

Synchronisation bidirectionnelle entre l'input utilisateur et le modèle :

Dans le fichier TypeScript
username = "John Doe";

Dans le fichier HTML:
<input [(ngModel)]="username" />

⚠️ il est faut importer FormsModule dans votre module pour pouvoir utiliser ce type de binding !

Résultat :
=> {{ username }}
